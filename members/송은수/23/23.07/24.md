# Study

내가 지금 무엇이 궁금하냐 분석해봤더니, REST API 그리고 json형태라는 것이 무엇인지 궁금해 한다.  
그래서 MERN 강의를 들어보며 벡엔드와 db를 일부 공부해보고자 한다.

## MERN

mongodb, expressjs, reactjs, nodejs의 앞글자를 따온 것이다.  
일반적으로 간단한 풀스택을 만들때 많이사용하는 방법.

### REST API, json

react에서 UI,유틸리티 등 프론트엔드를 만들고 nodejs에서 벡엔드를 담당한다.  
db에는 데이터들이 저장되어 있는데, react에서 한번에 여기에 접근한다면 보안성에 문제가 생긴다.  
react는 필요한 데이터를 벡엔드에 요청을 보내고, 벡엔드는 이를 받아 db에서 데이터를 꺼내온다. 이 과정이 REST API.
벡엔드에서 프론트엔드에 자료를 보내줄 때 특정 형식이 필요할텐데, 이 형식이 json이다.
