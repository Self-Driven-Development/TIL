# TIL(Today I Learned)

> 매일 공부한 내용을 **`다같이`** 정리하는 레포지토리

## 작성 규칙

자세한 내용은 [가이드라인](https://github.com/Self-Driven-Development/TIL/tree/main/GUIDELINE.md)을 참조하세요.

## 작성 목적

- 공부한 내용을 정리하는 습관을 기를 수 있어요
- 공부한 내용을 다른 사람과 공유할 수 있어요
